Opis rozwiązywanego problemu:

Przedmiotem zainteresowania jest stożek wrzucony do wody. Należało tak dobrać gęstość, aby stan równowagi odpowiadał zanurzeniu do 2/3 wysokości stożka.

Raport projektu zawiera:
- opis problemu - rysunek wraz z siłami
- zamodelowane równania różniczkowe ruchu
- rozwiazanie numeryczne układu równań
- odpowiednie wykresy
